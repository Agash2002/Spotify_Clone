# Spotify Clone (HTML & CSS Only)

This project is a static clone of Spotify's web interface, focusing on its layout and visual design. It has been created using only HTML and CSS.

## Features
- **Home Page**: Replicates the design of Spotify's landing or main dashboard.
- **Playlist Page**: Mimics a playlist page layout, including song listings.
- **Custom Styling**: Includes custom buttons, navigation bar, and hover effects.

## Technologies Used
- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout, including Flexbox .
- **Frameworks/Libraries**: [Optional: Bootstrap, Tailwind CSS]
